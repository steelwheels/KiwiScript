# Point Primitive
The *Point* is data structure to store 2D point.

## Constructor
````
let point = Point(x, y) ;
````

### Parameters
|Name       |type       |Description          |
|:--        |:---       |:---                 |
|x          |Double     |X position           |
|y          |Double     |Y position           |

If `x` or `y` is not numerical value,
the `undefined` value will be returned.

## Object
### Properties
|Name       |type       |Description          |
|:--        |:---       |:---                 |
|x          |Double     |X position           |

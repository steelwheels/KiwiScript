# Pipe operation

## `Pipe` class
### `open` class method
Allocate Pipe object. It has input and input file.
````
var pipe   = Pipe.open() ;
````
#### Parameters(s)
None
#### Return value
The *PipeObject* object allocated by this method.

## `PipeObject` class
#### `input` property
The input file (instance of *FileObject*).
#### `output` property
The output file (instance of *FileObject*).

# The data representation of primitive values

## Core graphics
### Point
#### Data representation
````
{
        class:  "point",        // string
        x:      0.0,            // float
        y:      0.0             // float
}
````

#### Data type
[CGPoint](https://developer.apple.com/documentation/coregraphics/cgpoint)

### Size
#### Data representation
````
{
        class:  "size",         // string
        width:  0.0,            // float
        height: 0.0             // float
}
````

#### Data type
[CGSize](https://developer.apple.com/documentation/coregraphics/cgsize)

### Rect
#### Data representation
````
{
        class:  "rect",         // string
        x:      0.0,            // float
        y:      0.0,            // float
        width:  0.0,            // float
        height: 0.0             // float
}
````

#### Data type
[CGRect](https://developer.apple.com/documentation/coregraphics/cgrect)

# Data 

## Data operations
### `Allocate2D`: Allocate 2D array
````
let array = Array2D(width, height, initval) ;
````

|Parameter(s)   |Type   |Description                |
|:--            |:--    |:--                        |
|width          |Int    |Width of 2D array          |
|height         |Int    |Height of 2D array         |
|initval        |Any    |Initial value of all elements  |

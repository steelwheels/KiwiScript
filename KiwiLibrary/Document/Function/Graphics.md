# Graphics Data Structures And Functions
## `Point`
### Data structure
````
{
        x:      <x-position>,   // Float
        y:      <y-position>    // Float
}
````

### Constructor
````
let newpt = Point(x, y) ;
````

## `Size`
### Data structure
````
{
        width:  <width>,        // Float
        height: <height>        // Float
}
````

### Constructor
````
let newsz = Size(width, height) ;
````

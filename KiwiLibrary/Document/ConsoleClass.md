# Console Operation
The *Console* class is used to print log message into the terminal.

## Console Class
Print log message into terminal.
### `log` class method
````
Console.log(<message>)
````
### Parameter(s)
|Parameter    |Type   |Description                    |
|:---         |:---   |:---                           |
|message      |String |Log message string             |

### Return value
none

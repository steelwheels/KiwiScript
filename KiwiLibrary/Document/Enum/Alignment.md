# Alignment Type

The `Alignment` object has the table of kinds of alignment.

## Properties
Following properties are used to specify the alignment.
Every properties has signed integer values:
* `Alignment.left`
* `Alignment.center`
* `Alignment.right`
* `Alignment.top`
* `Alignment.middle`
* `Alignment.bottom`
